# Medical Profile App

A comprehensive React Native application for managing personal health information, tracking medical data, and maintaining health records.

## Features

-   Patient Profile Management
-   Health Insights Dashboard
-   Medication Tracking
-   Appointment Management
-   Document Storage
-   Emergency Information
-   Health Articles Feed
-   Medication Side Effect Reporter
-   Mental Health Mood Tracker
-   Lab Results Visualizer

## Prerequisites

-   Node.js >= 16
-   React Native CLI
-   Xcode (for iOS development)
-   Android Studio (for Android development)

## Installation

```bash
# Install dependencies
npm install

# Start the application
npm run start

# Start Android
# Start the application
npm run android
```

## Development

```bash
# Run on iOS
npm run ios

# Run on Android
npm run android

# Run tests
npm run test
```
